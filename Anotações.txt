Introdução a Git/Github

Comandos Úteis

mkdir - criação de pasta
ls - lista os arquivos
cd - entra em determinada pasta
cd.. - retrocede uma pasta
pwd - imprime o nome do diretório local
eval - executar argumentos como um comando shell
cat - permite a criação de novos arquivos de texto de forma rápida

ssh-add id_ed25519 - cria chave ssh

git init - cria um novo repositório do Git
git clone - clona os repositórios para sua máquina
git status - mostra o status do seu repositório
git commit -m "alguma mensagem" - comita seu repositório para subir pra nuvem
git add - adiciona novos arquivos ao seu repositório
git push origin master/main - coloca o novo repositório no Github

. ou -a - utilizado para usar todos os arquivos do diretório
